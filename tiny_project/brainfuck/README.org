* BrainFuck Interpreter

| 指令 | 含义                                                          | 等价的C代码       |
|------+---------------------------------------------------------------+-------------------|
| >    | 指针加一                                                      | ++ptr;            |
| <    | 指针减一                                                      | --ptr;            |
| +    | 指针指向的字节的值加一                                        | ++*ptr;           |
| .    | 输出指针指向的单元内容（ASCII码）                             | putchar(*ptr);    |
| ,    | 输入内容到指针指向的单元（ASCII码）                           | *ptr = getchar(); |
| [    | 如果指针指向的单元值为零，向后跳转到对应的]指令的次一指令处   | while (*ptr) {    |
| ]    | 如果指针指向的单元值不为零，向前跳转到对应的[指令的次一指令处 | }                 |
